# laravel-core

## Возможности:
- Получение массива Dto-объектов с данными из файла конфигурации приложения на laravel.
- ...
## Работа с пакетом:
- Для получения массива Dto-объектов с множеством данных из файла конфигурации необходимо использовать статичный метод createArray класса DtoFactory. Первым параметром передается массив строк, полученный методом config(). Вторым параметром передается строка пути к Dto-классу.
- Для получения одного Dto-объекта с единичными данными из файла конфигурации необходимо использовать статичный метод dtoOfArray класса DtoFactory с аналогичными параметрами.
